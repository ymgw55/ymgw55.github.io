How to use Academic

Hiroaki Yamagiwa
Jan 19, 2020 2 min read settings

We can make our own website on http://[username].github.io/ for free if you have your GitHub account.

If you want to know more detail, please go to GitHub Pages.

This page is made by Hugo, which is one of the most popular OSS static site generators.

Hugo has many kind of themes and I choosed Academic. Academic is very functional, that is, there are many choices to customize, however, it is sometimes too much for me. That is why I take notes for some practical settings for my own in this page.

By using Academic, we can make pages from markdown, without setting HTML or CSS, because generators generates such kinds of files automatically.

Setting for menu bar

Go to config/_default/menus.toml, and we can change setting for display, sequence, and link about items on menu bar such as Home, Posts, and so on.

Setting for bibliography

Go to content/authors/admin/_index.md.

Setting for home page

Go to content/home/*.md, and then we can change setting for display and so on. For example, in the case of posts.md, that markdown file interact with content/post/, and we can decide how many directories display.

Setting for [username].github.io/*

For example, in the case of [username].github.io/post, go to `content/post/_index.md, and then we can change setting for title or subtitle on home page.

How to make new pages

If you want to know detail, go to this official document. For example, in the case of making new post, type hugo new --kind post post/my-article-name in terminal, and then the new folder whose name is my-article-name is made automatically, so we can write contents on index.md in the folder. In this case, to show a new page in home, you go to content/home/posts.md and ensure active = true.

How to write technical content

Go to official demo pages.

Setting for contact

Go to config/_default/params.toml, and then we can change setting for our telephone number, our street address and so on.

How to check changes

Type hugo server in terminal, and then hugo server starts and we can go to localhost:1313 for checking changes.
